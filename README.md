## Sobre o projeto

O Eco Vigilante é um Projeto Integrador desenvolvido durante o primeiro semestre da graduação de Tecnologia em Sistemas para Internet, com o objetivo de criar um site institucional para uma ONG fictícia voltada à conscientização e combate ao garimpo ilegal no Brasil.

A proposta busca informar a população sobre os impactos ambientais, sociais e na saúde pública causados pela atividade ilegal, além de incentivar a preservação dos recursos naturais e ampliar o acesso a canais oficiais de denúncia.

O projeto também estabelece relação com os Objetivos de Desenvolvimento Sustentável (ODS), promovendo reflexão sobre sustentabilidade, proteção ambiental e responsabilidade social.

---

## Objetivos

* Conscientizar sobre os impactos do garimpo ilegal
* Informar a população através de uma interface acessível e intuitiva
* Incentivar denúncias através de canais oficiais
* Relacionar o tema aos Objetivos de Desenvolvimento Sustentável (ODS)
* Desenvolver uma interface web responsiva e semântica
* Aplicar conceitos de UI Design e experiência do usuário
* Trabalhar organização visual e hierarquia de informações
* Desenvolver um projeto web utilizando HTML, CSS e JavaScript

---

## Sobre o Eco Vigilante

O Eco Vigilante é uma ONG fictícia criada para representar iniciativas de conscientização ambiental voltadas à preservação dos recursos naturais brasileiros.

A proposta visual do projeto utiliza elementos gráficos e cores associadas à natureza, sustentabilidade e proteção ambiental, buscando transmitir seriedade, informação e impacto social.

O site apresenta conteúdos educativos, dados informativos e direcionamentos para denúncia, reforçando a importância do combate ao garimpo ilegal e da preservação ambiental.

---

## Estrutura e experiência do usuário

O projeto foi desenvolvido priorizando acessibilidade visual, organização estrutural e navegação intuitiva.

Durante o desenvolvimento da interface, foram considerados fatores como:

* Legibilidade textual e hierarquia visual
* Contraste adequado entre cores e elementos
* Consistência da identidade visual
* Navegação intuitiva e objetiva
* Organização semântica do conteúdo
* Responsividade para diferentes dispositivos
* Estrutura baseada em Web Standards

Além disso, a página foi planejada para direcionar o usuário de forma clara às informações sobre conscientização ambiental, impactos do garimpo ilegal e canais oficiais de denúncia.

---

## Identidade visual

O projeto foi desenvolvido com uma identidade visual moderna e acessível, utilizando cores inspiradas em elementos naturais e ambientais.

A interface foi planejada no Figma, priorizando organização visual, responsividade e boa experiência do usuário em diferentes dispositivos.
